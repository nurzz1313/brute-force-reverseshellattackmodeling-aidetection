Моделирование атак (Brute Force, Reverse Shell) с детектированием на основе ИИ
Дипломный проект, в котором атаки Brute Force и Reverse Shell моделируются в контролируемой виртуальной лаборатории, а детектор на основе ИИ (Random Forest) классифицирует возникающие события и реагирует в реальном времени: генерирует оповещения и блокирует источник атаки.

Дисклеймер
Проект создан исключительно для академического исследования и предназначен для запуска только внутри изолированной виртуальной лаборатории (host-only сеть в VirtualBox). Обзор
Традиционные сигнатурные и правил-ориентированные средства защиты плохо справляются с новыми или модифицированными атаками. В проекте реализован сквозной (end-to-end) пайплайн, который:
Генерирует реалистичные события аутентификации и загрузки файлов из локального веб-приложения.
Воспроизводит против него два сценария атак (Brute Force и Reverse Shell).
Собирает сырые логи и преобразует их в датасет, готовый для машинного обучения.
Обучает классификатор Random Forest размечать события как benign, bruteforce или reverse_shell (подозрительная загрузка).
Запускает детектор реального времени, который классифицирует новые события, пишет оповещения и блокирует IP-адрес атакующего.
Архитектура лаборатории
Стенд работает в Oracle VirtualBox на host-only сети:


Веб-приложение содержит страницу входа (события аутентификации) и страницу загрузки файлов после авторизации (файловые события). Логи пишутся без меток — разметка выполняется офлайн на этапе подготовки датасета.
Как это работает


Классы детектирования
benign — нормальная активность пользователя
bruteforce — повторяющиеся неудачные входы, сконцентрированные во времени, иногда завершающиеся успешным входом
reverse_shell — подозрительная загрузка файла (опасное расширение / несоответствие MIME), связанная с пост-компрометацией
Примеры признаков
Аутентификация: число неудачных входов с одного IP в окнах 60с / 300с, число неудачных попыток на пользователя, время с предыдущего события, успешный вход после серии неудач.
Загрузка: расширение файла, размер, MIME-тип, флаг подозрительного расширения, флаг несоответствия MIME, длина и энтропия имени файла, загрузка после серии неудачных входов.
Стек технологий
Python (scikit-learn, pandas) — предобработка, обучение, детектор реального времени
Random Forest — многоклассовый классификатор
PHP — локальное веб-приложение со входом / загрузкой и проверкой блок-листа
Ubuntu + Kali Linux на Oracle VirtualBox — лабораторная среда

Быстрый старт
Требования
Oracle VirtualBox с целевой ВМ Ubuntu и атакующей ВМ Kali на host-only сети
Python 3.10+
PHP 8.x на целевом хосте
Установка
git clone git@github.com:nurzz1313/brute-force-reverseshellattackmodeling-aidetection.git
cd brute-force-reverseshellattackmodeling-aidetection
pip install -r requirements.txt
Запуск

1. Запустить веб-приложение на целевом хосте
php -S 0.0.0.0:8000 -t webapp/
 
2. Подготовить датасет из собранных сырых логов
python prepare_dataset.py
 
3. Обучить модель
python train_model.py
 
4. Запустить детектор реального времени
python detector.py


Затем запустите сценарии атак с хоста Kali против цели и наблюдайте, как детектор генерирует оповещения и блокирует IP-адрес источника.
Результаты
Детектор распознаёт аномалии аутентификации Brute Force и подозрительные загрузки во время живого исполнения, генерирует структурированные записи оповещений и применяет блокировку IP через веб-приложение. Полные метрики (accuracy, precision, recall, F1) и матрицы ошибок приведены в пояснительной записке диплома.
